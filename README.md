# はかルンです
## 概要
飲料の糖質とカフェインを一目で把握。一日の摂取量を見える化するアプリ。
<div align="center">
    <img src="./img/home.png" width="30%">
</div>

## 機能一覧
- 飲料のバーコードを読み取って、糖質とカフェイン量を確認できる
- 一日の摂取量を計算して、可視化できる
